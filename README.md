# Amazon India Scraper

Scrapes Amazon.in search results with auto-pagination.

## Features
- Dynamic keyword search
- Auto-pagination (all pages)
- Clean rating format (4.2/5)
- Reviews count extracted
- Product URL + Image URL
- CSV + JSON export
- Rate limiting (anti-bot)

## Results
- 328 products scraped for phone
- 20 pages automatically

## Tech Stack
Python, Requests, BeautifulSoup, Pandas, Regex

## Usage
pip install requests beautifulsoup4 pandas
python amazon_scraper.py
